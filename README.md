# Ethical UX

A React + Storybook project exploring dark patterns and better alternatives through interactive UI examples.

Instead of only describing manipulative UX, this project shows it in action: each pattern starts with the bad version first, then lets you switch to a more respectful implementation.

## Overview

Ethical UX is a small pattern library focused on trust, clarity, and user control.

Using Storybook as the primary interface, the project presents common deceptive flows such as cookie consent, hidden fees, confirmshaming, forced account creation, and hard-to-cancel interactions. Each example is designed to make the problem visible first, then demonstrate how the same experience can be rebuilt in a clearer and more user-respecting way.

## Features

- Storybook-driven pattern explorer
- Dark pattern vs better pattern comparisons
- React component examples in isolation
- UX-focused notes on what changes and why it matters
- A foundation for ethical UI experiments and discussion

## Tech Stack

- React
- TypeScript
- Storybook

## Goal

This project exists to make ethical interface design concrete.

It is part pattern library, part teaching tool, and part component showcase — built to explore how small UI decisions can either manipulate users or respect them.
